# ASM - резидент
## Программа-резидент на ассемблере для dosbox. (regs_on_screen.asm)

Перед прерыванием клавиатуры (09h) проверяет нажата ли одна из комбинаций клавиш Ctrl+S или Ctrl+A.

### Ctrl + S
При нажатии на комбинацию клавиш ctrl+S в левом верхнем углу появляется рамка со значениями 13 регистров (см. изображение)
<img width="1040" height="676" alt="image" src="https://github.com/user-attachments/assets/3b57dcc6-50f6-4f5a-8491-8399239743db" />
Значения регистров обновляются каждый тик таймера, как и рамка - то есть, она всегда на экране до нажатия клавиши Ctrl+A.

### Ctrl + A
При нажатии на комбинацию клавиш Ctrl+A рамка закрывается. 

Для верного закрытия рамки реализована тройная буфферизация - информация буффера, где хранится состояние экрана до вывода рамки (save_buffer), обновляется каждый тик таймера, путем сравнения видеопамяти и буффера текущего состояния экрана (draw_buffer).

После нажатия на нужную клавишу (Ctrl+A) информация из save_buffer выводится на экран, закрывая таким образом рамку.

Такой способ не гарантиует правильного закрытия рамки в любой ситуации. 


## Программа-тестировщик резидента. (resident_test_programm.asm)

Кладет в регистры ax-ss "яркие" значения, для проверки правильности ввывода значений регистров.
В регистре al нахоидтся последний код нажатой на клавиатуре клавиши.

В бесконечном цикле проверяет, нажата ли клавиша esc (сравнением al с кодом клавиши esc). Если нажата, программа останавливается.
